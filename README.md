# ⚡ Daily Task Hub

> A modern, minimal, and visually engaging productivity dashboard to access your daily tools in one place.

---

## 🚀 Overview

**Daily Task Hub** is a lightweight, single-page web application designed to centralize frequently used tools such as AI platforms, developer resources, and learning websites into a clean and interactive interface.

It focuses on:
- Speed ⚡
- Simplicity 🧩
- Visual Experience 🎨
- Productivity 📈

---

## ✨ Features

### 🎨 Modern UI Design
- Dark gradient background
- Glassmorphism card layout
- Smooth animations and transitions

### 🌌 Interactive Effects
- Floating particle background animation
- Mouse-tracking glow effect
- Hover scaling and rotation effects

### ⚡ Fast & Lightweight
- No frameworks or libraries
- Pure HTML, CSS, and JavaScript
- Instant loading

### 📱 Responsive Design
- Works across desktop, tablet, and mobile devices

### 🔗 Quick Access Dashboard
Includes shortcuts to:
- AI tools (ChatGPT, Gemini, Claude, Grok)
- Developer tools (GitHub, Supabase, Netlify)
- Learning platforms (Coursera, SoloLearn)
- Productivity apps (Gmail, LinkedIn, YouTube)
